# KnotMaster Pro 🪢

[![License](https://img.shields.io/badge/license-Proprietary-red.svg)](LICENSE)

Interactive knot tying tutorial with **140+ knots**, step-by-step guides, achievements system, confetti celebrations, quiz mode, progress tracking, and PWA support.

## Features ✨
- 🎯 **140+ knots** categorized by difficulty & use case
- 📱 **PWA ready** - installable, offline-first  
- 🏆 **Achievements** with confetti effects
- 🔍 **Advanced filtering** (difficulty, category, search)
- 📊 **Progress tracking** with gradient SVG bars
- 🎨 **Glassmorphism UI** with neon gradients & animations
- 🌙 **Dark/Light theme**
- 🧠 **Quiz mode** (coming soon)

## Live Demo
- [GitHub Pages](https://bingkahu.github.io/knot-tying-101)
- Local: `npx live-server .`
